This page captures media keys (next/prev/stop/pause) by key name string.

Hot keys cheat sheet
  • H ... Go to Home page.
  • C ... Go to Counter page.
  • F ... Go to Fetch data page.
  • Ctrl ... Double tap to go to Home.
  • @ ... (only "Home") change greeting message. (for testing AddByKey())
  • U ... (only "Counter") Increment counter.

The hot key H will work even if input element has focus.
